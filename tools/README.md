## Was sind MCP Server?

Das [Model Context Protocol](https://modelcontextprotocol.io/) ermöglicht Claude Code, mit externen Services zu kommunizieren. MCP Server erweitern Claude's Fähigkeiten um Tools wie Datenbankzugriff, API-Calls, Browser-Steuerung und mehr.
