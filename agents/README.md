# Agents

Custom Agent Konfigurationen für spezialisierte Aufgaben.

## Was sind Agents?

Claude Code kann spezialisierte Sub-Agents starten, die bestimmte Aufgaben autonom erledigen — z.B. Code-Review, Testing oder Recherche.
