# CLAUDE.md Beispiele

Beispiel-CLAUDE.md Dateien für verschiedene Projekttypen. Nutze sie als Startpunkt für deine eigene CLAUDE.md.

## Was ist eine CLAUDE.md?

Die `CLAUDE.md` ist die zentrale Konfigurationsdatei für Claude Code. Sie definiert Kontext, Regeln und Workflows für dein Projekt. Claude liest sie automatisch bei jedem Start.

## Inhalt

| Beispiel | Für wen? |
|----------|----------|
| [fullstack-webapp.md](./fullstack-webapp.md) | Next.js + Prisma + Auth Projekte |
| [solo-saas.md](./solo-saas.md) | Solopreneure mit Supabase + Stripe |
| [community-orga.md](./community-orga.md) | Meetup/Community-Organisatoren |

## Tipps für gute CLAUDE.md Dateien

1. **Projekt-Kontext zuerst** — Was ist das Projekt, welcher Stack?
2. **Regeln klar formulieren** — Was darf Claude, was nicht?
3. **Commands auflisten** — Welche npm/dev-Befehle gibt es?
4. **Architektur skizzieren** — Ordnerstruktur, Patterns
5. **Aktuell halten** — CLAUDE.md ist ein lebendes Dokument
