## Was ist das?

Eine kuratierte Sammlung von Ressourcen rund um Claude Code — zusammengetragen von der Münchner Community. Hier findest du praktische Beispiele, die du direkt in deinen Workflow übernehmen kannst.

---

## Inhalt

| Ordner | Was findest du dort? |
|--------|---------------------|
| [`skills/`](./skills/) | Claude Code Skills (Slash-Commands) |
| [`agents/`](./agents/) | Custom Agent Konfigurationen |
| [`claude-md/`](./claude-md/) | Beispiel-CLAUDE.md Dateien für verschiedene Projekte |
| [`prompts/`](./prompts/) | Nützliche Prompts für wiederkehrende Aufgaben |
| [`tools/`](./tools/) | MCP Server, Configs und hilfreiche Scripts |

## Schnellstart

```bash
# Repo klonen
git clone https://github.com/Claude-Code-Meetup-Munich/community-resources.git

# Beispiel: Einen Skill installieren
cp community-resources/skills/commit.md ~/.claude/commands/commit.md

# Beispiel: CLAUDE.md als Inspiration nutzen
cat community-resources/claude-md/fullstack-projekt.md
```

## Beitragen

Wir freuen uns über Beiträge! Siehe [CONTRIBUTING.md](./CONTRIBUTING.md) für Details.

**Kurzversion:**

1. Fork dieses Repo
2. Erstelle einen Branch (`git checkout -b mein-beitrag`)
3. Füge deine Ressource im passenden Ordner hinzu
4. Erstelle einen Pull Request

Jeder Beitrag wird von einem Organizer reviewed.

## Qualitätsstandards

- **Funktioniert**: Jede Ressource wurde getestet
- **Dokumentiert**: Kurze Beschreibung was es tut und wie man es nutzt
- **Eigenständig**: Keine externen Abhängigkeiten ohne Erklärung
- **Respektvoll**: Keine Inhalte die gegen unseren Code of Conduct verstoßen
